# 🚀 ATS Portal (Applicant Tracking System)

A Full Stack Applicant Tracking System built using the MERN Stack.

## 📌 Features

### 👨‍💼 Recruiter

- Register / Login
- JWT Authentication
- Create Company
- Edit Company
- Upload Company Logo
- Create Jobs
- Edit Jobs
- Delete Jobs
- View Applicants
- Recruiter Dashboard
- Notifications
- Profile Management

### 👨‍🎓 Student

- Register / Login
- Browse Jobs
- Search Jobs
- Filter Jobs
- Apply for Jobs
- View Applied Jobs
- Student Dashboard
- Profile Management
- Resume Upload

# ⚙ Installation

## Clone Repository

```
git clone <repository-url>
```

---

## Backend

```
cd server

npm install

npm start
```

---

## Frontend

```
cd client

npm install

npm run dev
```

---

# Environment Variables

Create `.env`

```
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key
```
